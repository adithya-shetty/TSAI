# Movie Review Classification with Glove Word Embeddings using simple neural network
Classify movie reviews into positive and negative with GloVe embeddings.

## Description
  - Dataset : IMDB dataset
  - GloVe embedding dim : 50
  - Train samples : 8000
  - Val samples : 10000
  - Max words : 10000
